Driver Gaze And Engagement Estimation Project

This project implements a real-time gaze tracking system using a single webcam to estimate driver engagement in driving simulation environments.

## Features

- Real-time gaze tracking using webcam
- Face landmark detection with MediaPipe
- Logs gaze data to CSV files
- Modular and extensible codebase

## Installation

- Clone the repository
- Install dependencies from requirements.txt
- Run the main script in the src directory

## Usage

- Run `python src/mediapipe_gaze.py` to start gaze tracking
- Check the `logs/` folder for output CSV files
